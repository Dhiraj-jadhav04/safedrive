# SafeDrive AI - Final Year Project
This is a Python-based Machine Learning and Computer Vision application for real-time drowsiness and fatigue detection.

## Features
- AI-based EAR (Eye Aspect Ratio) monitoring.
- Deep Learning Face Landmarker (MediaPipe).
- Real-time MAR (Mouth Aspect Ratio) for yawning detection.
- PPG-based heart rate estimation.
- automated safety report generation.

## Technical Stack
- **Languages**: Python
- **Libraries**: Streamlit, OpenCV, MediaPipe, NumPy, Plotly
- **AI Models**: MediaPipe Face Landmarker (face_landmarker.task)
